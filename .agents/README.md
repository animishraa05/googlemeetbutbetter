## 🎨 Design Style: Neobrutalism

**MANDATORY for all UI:**

```
✅ Bold borders (3px+)
✅ Vibrant colors (#FF6B6B, #4ECDC4, #FFE66D)
✅ Hard shadows (4px 4px 0px #000)
✅ Sharp corners (0-4px)
✅ Heavy typography (700-900 weight)

❌ No soft shadows
❌ No muted colors
❌ No rounded corners > 4px
❌ No minimalism
```

## ✅ Quality Checks

**Before accepting design:**
- [ ] Neobrutalism style visible?
- [ ] All colors specified (hex)?
- [ ] All typography defined?
- [ ] All states described?
- [ ] Responsive behavior specified?

**Before accepting code:**
- [ ] Matches design 95%+?
- [ ] No TypeScript errors?
- [ ] API integrated?
- [ ] All states handled?
- [ ] Tested with backend?

---

## 🚨 If AI Makes Mistakes

**AI skipping design?**
```
Stop. Read @.agents/workflow.md
Start with Design Agent phase.
```

**AI not using neobrutalism?**
```
This doesn't match neobrutalism.
Review @.agents/design.agent.md
Use: bold borders, vibrant colors, hard shadows.
```

**AI not integrating backend?**
```
Integrate with frontend/src/api.ts
Test with backend running.
```
